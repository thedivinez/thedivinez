Scan QR Code

Save WhatsApp Contacts


This app allows you to download unsaved contacts from your whatsapp and give you a contacts.vcf file which you can use to save the contacts to your phone.

1. Enter Name Suffix
2. Click the Start button
3. Open WhatsApp and scan QR Code
4. Click the download button
5. Copy the contacts.vcf file to your phone
6. Open the file in your phone and import contacts
7. All done you can see the contacts in your phonebook

Optional, describes how the contact name ends e.g thedivinez_CONT1
